## استقرار روی GitHub Pages
1. روی اکانتت یک repo جدید بساز به نام **wordmatch** (پابلیک باشه).  
2. تمام فایل‌های بالا رو داخل ریپو بریز.  
3. Settings → Pages → Source: Deploy from a branch → main → Save.  
4. بعد از ۳۰ ثانیه آدرست می‌شه:  
   `https://mrFarzadProgrammer.github.io/wordmatch/`  
5. داخل `@BotFather` برو:  
   `/newgame` → نام بازی → آدرس بالا رو بده.  
6. برای تغییر دیتابیت کافیه فایل `db.json` رو ادیت کنی و commit کنی – بلافاصله اعمال می‌شه!  
7. ادمین‌پنل: `https://mrFarzadProgrammer.github.io/wordmatch/admin.html` پسورد: `1234`
